Excel file template updated with your specific labware. An example can be downloaded here. NOTE: The program will work regardless of how many rows or columns you give it, it just ignores additional rows past what fits on your plate and it will add as many reagents to dispense as you like. HOWEVER, the mantis can only actually dispense 6 at a time. IF you wish to ignore any rows or columns create a column named "#". Then any column names starting with "#" will be ignored and any rows with a "#" in the "#" column will be ignored. You can try the test file to see how it works.
